Semana 2

Clase virtual - Miércoles 06/09

Mediante nuestra delegada, el profesor nos mandó documentos para descargar, estos serian el programa que usariamos para crear la bitácora y también la app donde la compartiriamos que en este caso fue Netlify.

¿QUÉ ES UNA BITÁCORA?

Una bitácora es un registro cronológico con el objetivo de explicar el proceso de eventos o actividades importantes.

Entonces, bajo ese concepto nosotros creariamos una en base a los programas que el profesor nos mandó a descargar. Uno de ellos es Visual Studio Code, que es donde hariamos la programación para nuestra página web.

Visual Studio Code es un editor de códigos y como otros VS Code se apega a otros interfaces de desarrollo de código. Aquí nos enseñó a cómo usar el programa, primero comenzaríamos con cosas basicas a como poner el título, subtitulo, párrafo e imágenes.

Clase presencial - Sábado 09/09

En esta clase el profesor nos enseñó a como usar el programa, colocando como primera instancia la palabra "DOCTYPE HTML" que sería el tipo o versión del documento.

Aprendimos a adjuntar imágenes con la abreviatura "img", audios con la abreviatura "audio" y videos con la abreviatura "video", a estos podiamos modificar el ancho y alto usando las palabras "width" y "legth" respectivamente. También para agregar un vínculo o dirección web usabamos el comando "a".

Para poder visualizar nuestra página web, primero gruadamos la programación con "Ctrl + S" y seguidamente abriamos "index.html".
