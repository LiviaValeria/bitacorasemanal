Semana 3

Clase virtual - Miércoles 13/09

En esta clase, el profesor se adentró más a la explicación sobre HTML y CSS, el profesor fue bien paciente con nosotros y nos volvió a enseñar como crear un documento en Visual Studio Code. Nos brindó mayor información sobre como vincular un archivo CSS con el documento, tanmbién nos explicó sobre los componentes de HTML que fueron los siguientes:

Cabecera - Es lo que contiene la información sobre el documento, aquíencontramos el título de la página y enlaces de archivos (CSS).

Encabezado - Son los elementos h1, h2, h3, etc, estos se usan para definir la fuente de título, subtítulo, entre otros.

Párrafos - Aquí se usa el comando "p", en estos podemos escribir textos y dividirlos por cada que pongamos el comando.

Enlaces - Se usa el comando "a" para poder vincular algún link o página web.
